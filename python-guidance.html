Python Curriculum

Month 1: Python Fundamentals

Week 1: Introduction to Python

  • Overview of Python and its applications
  • Setting up Python environment (interpreter, IDEs)
  • Basic syntax, variables, and data types
  • Control flow: if statements, error handling
  • Writing modular code: functions and modules
  • Loops (for, while)
  • Scope and namespaces

Week 2: Data Structures and Algorithms

  • Lists, tuples, dictionaries, sets, and their manipulation
  • Introduction to algorithm analysis and complexity
  • Searching and sorting algorithms: linear search, binary search, bubble sort, insertion sort

Week 3: Some Advance Python Programming

  • List/Dict comprehensions
  • Decorators
  • Lambda
  • Ternary operator
  • map/reduce/filter/zip
  • Concurrency/Parallel programming

Week 4: Object-Oriented Programming (OOP) in Python

  • Understanding classes, objects, and inheritance
  • Encapsulation and abstraction
  • Polymorphism and method overriding

Week 5: Object-Oriented Programming (OOP) in Python

  • Working with files and directories
  • Introduction to popular Python libraries: NumPy, Pandas, and Matplotlib
  • Hands-on exercises and mini-projects using Python libraries

Week 6: Getting Started with Django

  • Setting up a Django project and understanding project structure
  • Introduction to Django ORM and models
  • Creating basic views and templates

Week 7: Advanced Django Concepts

  • Authentication and authorization in Django
  • Working with Django forms and form validation
  • Handling static files and media files in Django

Week 8: Django REST Framework (DRF) Basics

  • Introduction to RESTful APIs
  • Building APIs using Django REST Framework
  • Serializers, ViewSets, and routers in DRF

Week 9: Advanced Django and DRF

  • Customizing DRF serializers and views
  • Authentication and permissions in DRF
  • Testing Django and DRF applications

Month 3: Project Development and Preparation for Interviews

Week 10-11: Project Development

Final Project: Students choose from one of the following options:

  • E-commerce Website: Build a fully functional e-commerce platform using Django, including user authentication, product listings, shopping cart functionality, and payment integration.
  • Social Media Platform: Create a social media platform where users can register, create posts, follow other users, and engage in discussions.
  • Task Management System: Develop a task management system with features for creating, assigning, and tracking tasks, along with user authentication and permissions.

Students work on their chosen project under the guidance of instructors, incorporating all learned concepts.

Week 11: Interview Preparation

Prepare for technical interviews by revisiting concepts covered throughout the curriculum and practicing coding problems.
